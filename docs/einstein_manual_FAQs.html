1. Problems with the set-up of EINSTEIN

Q: Data base connection error

The installation of EINSTEIN was carried out with success. EINSTEIN starts, but a "connection error" with the einstein database occurs

A: First check if the einstein database is installed well in MySQL. If everything is ok, check the configuration of the parameters in the menu databases->database administration.
The default configuration is:

host: localhost
database name: einstein
user name: root
password: none (leave blank space)

Use "test connection" in order to see whether EINSTEIN can connect with the database. If there's any problem, check the configuration parameters.
