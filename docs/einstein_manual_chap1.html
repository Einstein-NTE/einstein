1.1. Installation of EINSTEIN V1.00a for Newcomers

Note: V1.00a is still an INTERNAL testing version !!!

If You already have successfully installed EINSTEIN V.001 (demo version of the Sabadell meeting) please jump to section 1.2.

1.1.1. Basic set-up previous to the installation of EINSTEIN

Einstein uses the following software packages that have to be installed PREVIOUSLY to the installation of EINSTEIN itself :
- MySQL (Version 5.0 or higher)
- Python (Version 2.5 or higher)
- Several Python complements:
    a) wxPython
    b) matplotlib
    c) numpy

HOW TO INSTALL THESE PACKAGES ?

Step 1: Create your einstein installation folder, e.g. C:\MyEINSTEIN.

Step 2: Copy the following files into C:\MyEINSTEIN and then unzip, in the order as specified (the second file will overwrite some of the old files. When asked, confirm overwriting files with OK):
- EINSTEIN_InternalRelease_001.zip (-> the installation package of the very first Meeting-Sabadell-Demo Installation)
- EINSTEIN_R1.00a.zip

Please CHECK: After this you should have (at least) the following files and folders in your "MyEINSTEIN" directory:
- EINSTEIN.bat
- install_mysql.bat
- install_python.bat
- install_einstein.bat
- check_Installation.bat
- folder "einstein"
- folder "installer"

Step 3: Set-up of MySQL (skip if you have already MySQL 5.0 or higher running, skip this section)

TAKE CARE: Uninstall any older Version of MySQL BEFORE installing the new Version !!! Parallel installation of two different MySQL Versions may lead to problems.

There are two possible ways to doing MySQL installation. We recomment use of the installer (option (a) ), but in case of any problem you can try it with option (b).

a) Run the file "install_mysql.bat" and follow the instructions given on the screen (NOTE: internet connection is necessary !!!)

b) go to: http://dev.mysql.com/downloads/mysql/5.1.html and download the Windows ZIP/Setup.EXE (x86)
(or one of the other versions, depending on your computer and operating system)

CHECK OF MySQL INSTALLATION (On Windows)

To check the correct installation of MySQL, You can open the Control Panel, and look for an icon called Administrative Tools.
Open that, and there is another icon called Services. Open that too.
You will see a list of programs running in your machine. The list is ordered alphabetically.
Look for MySQL. It should be marked as Running (or Initiated or something like that) and in the next column, it should be marked as Automatic.
These settings ensure that MySQL is started each time you start your computer.
If it is there, but not running, click on the line with the Right button and choose Start. This will start MySQL, but only this once.
To make it start always, choose Properties. There you can select Automatic.
If MySQL refuses to run, please first deinstall and then reinstall it.

Step 4: Set-up of  Python and complements

There are two possible ways to doing Python installation. We recomment use of the installer (option (a) ), but in case of any problem you can try it with option (b).

a) Run the file "install_python.bat" and follow the instructions given on the screen (NOTE: internet connection is necessary !!!)

b) Go to www.python.org/download and do it yourself.

The installers for the necessary complements you find at:

http://www.wxpython.org/
numpy: http://numpy.scipy.org/
matplotlib: http://matplotlib.sourceforge.net/

Install also these additional packages on your computer.

1.1.2. Installation of EINSTEIN

Step 1: Installation of the EINSTEIN database

There are two possible ways to doing the EINSTEIN database installation. We recomment use of the installer (option (a) ), but in case of any problem you can try it with option (b).

a) Run the file "install_einsteinDB.bat" and follow the instructions given on the screen.

b) Run EINSTEIN (see below) and use the Menu-Option:

Database -> Database Administration -> Restore Database

You should select the file "einstein_R1.00a.sql" in the folder <MyEINSTEIN>\einstein\sql

Step 2: Now have a try -> Run EINSTEIN

For any problems check the FAQ Section of this manual.

Step 3: Configuration of MySQL password:
If you configured MySQL with a password, you have to specify this password in the Einstein configuration:

There are two possibilities
a) edit manually the file einstein\GUI\einstein.ini and put the password there. You will see that there is no password, originally.

b) Use the option  "Database -> Database Administration" in EINSTEIN, and enter your password there.

  

1.2. Update from previous Versions

Step 1:  You can use your existing EINSTEIN installation folder, e.g. C:\<MyEINSTEIN> , or you can create a new one, e.g. C:\<MyEINSTEIN V1.00a>

Step 2: Copy the file "EINSTEIN_R1.00a.zip" into C:\<MyEINSTEIN> and then unzip.
(if you use the folder of an existing installation, this may overwrite some of the old files. When asked, confirm overwriting files with OK).

Please CHECK: After this you should have (at least) the following files and folders in your "MyEINSTEIN" directory:
- EINSTEIN.bat
- install_einsteinDB.bat
- folder "einstein"

Step 3: Install the update of the EINSTEIN database (-> section 1.1.2)

Step 4: Now have a try -> Run EINSTEIN

For any problems check the FAQ Section of this manual.
