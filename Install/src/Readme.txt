Steps necessary for compiling:

Add .netFramework, WindowsInstaller and MySQL Installpackages to the Prerequisites Folder.
Now you have two choices:

1.) Get the packages dotnetfx35.exe, mysql-essential-5.1.24-rc-win32.msi, WindowsInstaller-KB893803-v2-x86.exe

2.) Get newer/older packages and rename the packages in the installer script. (compatibility is not guaranteed)
